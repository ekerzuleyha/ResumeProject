# Projeye Genel Bakış
🏮 Full Stack CV & Portfolio Projesi
Bu proje, ASP.NET Core 6.0 ile geliştirilen tam kapsamlı bir kişisel CV ve portfolyo yönetim sistemidir.
Proje; ziyaretçilerin görebileceği modern bir kullanıcı arayüzü ile içeriklerin yönetildiği güçlü bir admin paneli olmak üzere iki katmandan oluşmaktadır.

🟢 Admin Paneli
Admin paneli, tüm içeriklerin eklenip düzenlenebildiği tam CRUD yapısına sahiptir. Dashboard'da proje, mesaj, okunmamış mesaj ve yetenek sayıları anlık olarak görüntülenir; son 5 mesaj özet kartlar halinde listelenir. İstatistikler sayfasında okunmuş/okunmamış mesaj oranları progress bar ile görselleştirilir. Mesajlar sayfasında durum takibi yapılabilir, referanslar onay/red durumuna göre yönetilebilir.

🟢 Kullanıcı Tarafı
Ziyaretçiler; hakkımda, yetenekler, deneyimler, hizmetler ve projeler gibi tüm bölümleri tek sayfalık modern bir tasarımda gezebilir. Portfolyo sayfasında kategoriye göre filtreleme yapılabilirken referans bölümünde onaylanmış yorumlar slider ile sunulmaktadır. İletişim formu üzerinden doğrudan mesaj gönderilebilmektedir.


# Kullanılan Teknolojiler

✔️ ASP.NET Core 6.0, Entity Framework Core

✔️ Tailwind CSS, Stitch AI

✔️ MSSQL / SQL Server

✔️ Material Symbols, özel dark tema

✔️ LINQ
